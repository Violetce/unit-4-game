Horses Collector!

You will be given a random number of foals as a target to reach at the start of the game.


There are four horses below - they are all broodmares. By clicking on a horse, you will add a specific amount of foals to your total number of foals.


You win the game by matching your total number of foals to the target number of foals; you lose the game if your total number of foals go above the target number of foals.


The foals added for each broodmare is hidden from you until you click on it.


Each time when the game starts, the game will change the target number of foals.
